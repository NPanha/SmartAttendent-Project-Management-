# Smart Attendance System

A modern, web-based student attendance system using QR code scanning technology with a beautiful, responsive interface.

## Features

### Student Features
- **QR Code Attendance**: Quick and easy attendance marking via QR codes
- **Attendance Tracking**: View attendance history and statistics
- **Course Schedule**: Interactive timetable with upcoming classes
- **Profile Management**: Update personal information and view academic details
- **Real-time Dashboard**: Overview of attendance stats and today's classes

### Technical Features
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop
- **Modern UI**: Gradient design with Tailwind CSS
- **Interactive Elements**: Smooth animations and transitions
- **QR Scanner**: Browser-based QR code scanning using HTML5 API
- **Charts & Analytics**: Visual attendance trends with Chart.js
- **Flash Messages**: User feedback with animated notifications

## Tech Stack

### Frontend
- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first CSS framework
- **JavaScript** - Interactive functionality
- **Chart.js** - Data visualization
- **Font Awesome** - Icons

### Backend
- **Flask** - Python web framework
- **PostgreSQL** - Database (configured but using mock data)
- **HTML5 QR Code Scanner** - Client-side QR scanning
